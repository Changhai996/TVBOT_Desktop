# Tree Beautify (Local TVBOT)

A locally deployable evolution of the TVBOT project.

## 🌟 Key Features (vs. Original TVBOT)
- **Local Data Privacy:** All phylogenetic trees and annotation data are processed locally. No data is uploaded to external servers.
- **Persistent State:** Drawing states and configurations are saved in the local `data/` directory.
- **No File Limits:** Local execution bypasses web-based upload size and timeout constraints.

## 🚀 Local Usage
1. **Install dependencies:** `pip install -r requirements.txt`
2. **Start server:** `python server.py`
3. **Open browser:** Go to `http://localhost:5000`
